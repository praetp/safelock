int safelock_fetch_status ( safelock_t  lock,
safelock_status_t *  status

Fetch current status of a Safelock.

Atomically fills a safelock_status_t struct with current details for the Safelock.

The Safelock will be automatically removed if the Safelock is "locked" and the owner process no longer exists. This may occur if the Safelock is not properly unlocked after a kernel crash or power failure.

Parameters:
[in]lockSafelock to query
[out]statusFilled with the current Safelock status
Return values:
0Success
EINVALThe Safelock has been removed and must be re-opened before further use
ENOTRECOVERABLEThe Safelock is broken and must be closed, then recreated before further use

Open and lock a Safelock.

Helper function combining safelock_open() and safelock_lock().

The lock file must be placed on a local filesystem that supports mmap(2). All processes using a Safelock must have read/write access to the lock file.

int safelock_open ( safelock_t *  lock,
const char *  filename,
mode_t  mode 
)

Open a Safelock.

The Safelock file will be created if it does not already exist and mode is non-zero.

The lock file must be placed on a local filesystem that supports mmap(2). All processes using a Safelock must have read/write access to the lock file.

Parameters:
[out]lockSafelock to initialise
[in]filenameSafelock filename to open/create
[in]modePermissions to use when creating a new lock file
Return values:
0Success
EACCESSPermission denied, or invalid permissions (missing owner R/W)
EEXISTUnable to create a unique Safelock
EINTRInterrupted by signal
EIOI/O error updating inode
ENODEVUnderlying filesystem does not support mmap
ENOMEMUnable to allocate memory for the Safelock
ENOSPCDisk full

May also return any errno values generated by:

  • open(2)
  • link(2)
  • write(2)
  • unlink(2)
  • mmap(2)

int safelock_remove ( safelock_t  lock)

Atomically remove a Safelock from the filesystem.

Tags the Safelock as "removed" and deletes the file. Any further operations on the Safelock (except safelock_close()) will return EINVAL.
